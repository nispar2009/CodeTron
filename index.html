    Docs



    Running your code is no big deal on CodeTron. Just click the button above the code field. Use to completely clear the output terminal. Use to reset all variables.

    You can create a maximum of five variables at a time. You can give each variable a string value. It is also possible to change the value of a variable using the same syntax as creating it. Use the following code.

    1=Hello, world!

    Here, variable 1 is being set to "Hello, world!". Do the same by replacing "1" with "2", "3", "4", or "5". Change the right side of the = to give the variable a different value.

    One can also return the value of a variable. Just surround the variable you want to return with <>.

    <1>

    This will return the value of variable 1. Replace 1 with the other variables to get their value.

    Custom text can also be returned using the below code.

    >Hello, world!

    This will return "Hello, world!" Change the part after the > to edit the message.

    Creating conditionals is also easy on CodeTron. First enter the name of the variable you want to compare. Then add a / and the after it, the value you want to compare it with. Then, enter the code that will run if the condition is true. However, watch out! One conditional can only have one output.

    1/Hello <1> >Hello

    This will return the value of variable 1 if it is equal to "Hello". And, irrespective of whether the condition is true or false, we will then get "Hello".

    Is is also possible to get user input on CodeTron. After typing this code, a text box appears in the terminal to enter an answer. After the ?, add the question the user should answer. This will be displayed as a tooltip in the text box. The user's input will be stored as a variable. Use the below code.

    1?Some random question

    Until an answer is entered in the terminal, one cannot run more code. Above, the part before the ? is the variable and the part after is the title of the question. Try changing the "1" in the code above to save the input in a different variable.
